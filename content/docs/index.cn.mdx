## 简介

**Pigsty** <span className="text-fd-muted-foreground text-sm">(/ˈpɪɡ staɪ/)</span> 是一个开箱即用，本地优先，开源免费的 <span className="text-blue-500 font-bold">PostgreSQL 数据库发行版</span>。

Pigsty 满足您 “一切皆用 PostgreSQL” 的需求，提供 PG 生态中最全面的扩展插件/内核分支支持；
并整合了 PG 生态开源组件与最佳实践，提供了自建企业级 PostgreSQL 服务所需的一切，
让您无需数据库专家，即可在几分钟内以云数据库 RDS 1/10 不到的纯硬件成本，自建专业级数据库服务。

## 安装

准备一台全新安装 <DynamicLink href="/[lang]/docs/prepare/linux">兼容Linux系统</DynamicLink> 的节点，然后使用以下命令下载 Pigsty：

```bash title="install"
curl -fsSL https://repo.pigsty.io/get | bash;
cd ~/pigsty; ./bootstrap; ./configure; ./install.yml;
```
