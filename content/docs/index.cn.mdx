Pigsty 提供了以下三样基础设施，帮助用户更好的利用 PostgreSQL 扩展生态系统的协同超能力：

- [**扩展目录**](/list) ： 查阅 **423** 个扩展插件的详细信息，找到满足您需求的插件
- [**扩展仓库**](/repo) ： 获取预先打包的 RPM/DEB 二进制包，在 10 大主流 Linux 平台上可用
- [**包管理器**](/pig) ： `pig` 命令行工具，屏蔽复杂度与操作系统差异，一键完成所有设置

## 扩展分类

<Cards>
    <Card icon={<Clock className="text-blue-500" />} title='TIME' href={"/cn/cate/time"}>
        时间时态扩展：时序数据库 TimescaleDB，时态数据库，版本控制表，定时任务，异步后台任务调度扩展。
    </Card>

    <Card icon={<Globe className="text-green-500" />} title='GIS' href={"/cn/cate/gis"}>
        地理空间扩展：PostGIS，地理空间数据类型、函数与索引，天空索引 Q3C，OGR FDW， 寻路算法，地理正/逆查询。
    </Card>

    <Card icon={<Brain className="text-purple-500" />} title='RAG' href={"/cn/cate/rag"}>
        AI与RAG扩展插件：向量数据库，DiskANN 向量索引，相似度度量函数集，库内机器学习与推理 pgml，等等。
    </Card>

    <Card icon={<Search className="text-orange-500" />} title='FTS' href={"/cn/cate/fts"}>
        全文检索扩展：ES 替代 pg_search，BM25，中文分词，欧洲语言分词字典 hunspell，模糊检索，2-gram/3-gram 索引。
    </Card>

    <Card icon={<ChartNoAxesCombined className="text-red-500" />} title='OLAP' href={"/cn/cate/olap"}>
        分析能力扩展：列式存储，DuckDB与外部数据源包装器，Parquet S3，数据冷热分级存储，分布式计算，透明分片，GPU加速。
    </Card>

    <Card icon={<Sparkles className="text-cyan-500" />} title='FEAT' href={"/cn/cate/feat"}>
        功能特性扩展：图数据库，Hyperloglog，Rum索引，GraphQL，JsonSchema，Hint，虚拟索引，增量物化视图，消息队列等等。
    </Card>

    <Card icon={<BookA className="text-indigo-500" />} title='LANG' href={"/cn/cate/lang"}>
        存储过程语言扩展：使用各种编程语言开发，调试，打包，分发，测试 PostgreSQL 存储过程：Java，Js，Lua，R，SH，PRQL…
    </Card>

    <Card icon={<Boxes className="text-teal-500" />} title='TYPE' href={"/cn/cate/type"}>
        自定义类型扩展：前缀树，语义版本号，SI单位，位图，无符号整型，高精度数值，有理数，哈希值，IP地址段，球面，RRULE等。
    </Card>

    <Card icon={<Wrench className="text-amber-500" />} title='UTIL' href={"/cn/cate/util"}>
        实用功能扩展：HTTP请求，GZIP压缩，JWT处理，邮件客户端，正则，字符编码，编码解码，加密解密等实用功能。
    </Card>

    <Card icon={<Variable className="text-pink-500" />} title='FUNC' href={"/cn/cate/func"}>
        标识聚合函数：ID生成器，各类聚合函数，摘要函数，数组处理函数，数学函数，统计量，伪随机，等等。
    </Card>

    <Card icon={<Landmark className="text-slate-500" />} title='ADMIN' href={"/cn/cate/admin"}>
        管理工具扩展：膨胀治理，脏读，检视缓冲区，数据目录，校验和，系统目录腐败检查，优先级管理，权限管理，语句准备，限制批量更新等。
    </Card>

    <Card icon={<Activity className="text-emerald-500" />} title='STAT' href={"/cn/cate/stat"}>
        监控统计扩展：AWR报告，可观测性指标，显示执行计划，查询统计信息，内存使用，配置变更，等待事件采样，慢查询日志，等等。
    </Card>

    <Card icon={<Shield className="text-red-600" />} title='SEC' href={"/cn/cate/sec"}>
        安全功能扩展：强制密码强度，阉割超级用户，密钥管理，商密算法，PII匿名处理，扩展白名单，审计日志，变更追溯，反病毒等等。
    </Card>

    <Card icon={<FileInput className="text-blue-600" />} title='FDW' href={"/cn/cate/fdw"}>
        外部数据源包装器：FDW开发框架 Wrappers，Multicorn，访问外部的 Mongo，MySQL，SQLite，HDFS，MSSQL，Oracle，DB2，……
    </Card>

    <Card icon={<Shell className="text-violet-500" />} title='SIM' href={"/cn/cate/sim"}>
        数据库兼容扩展：仿真其他 DBMS 的行为：MySQL，Memcache，Mongo，Oracle，Babelfish for Microsoft SQL Server……
    </Card>

    <Card icon={<Truck className="text-orange-600" />} title='ETL' href={"/cn/cate/etl"}>
        数据复制扩展：逻辑复制，逻辑解码，DDL复制，JSON/BSON/Protobuf 变更抽取，数据迁移，数据导入，数据比对等。
    </Card>
</Cards>
