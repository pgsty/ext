## Download

To add the required PGDG / PIGSTY upstream repository, use:

```bash tab="pig"
pig repo add pgdg -u    # add PGDG repo and update cache (leave existing repos)
pig repo add pigsty -u  # add PIGSTY repo and update cache (leave existing repos)
pig repo add pgsql -u   # add PGDG + Pigsty repo and update cache (leave existing repos)
pig repo set all -u     # set repo to all = NODE + PGSQL + INFRA  (remove existing repos)
```
```bash tab="pigsty"
./node.yml -t node_repo -e node_repo_modules=node,pgsql # -l <cluster>
```

## Install

[**Install**](/docs/pgsql/ext/install) this extension with:

```bash tab="pig"
pig ext install omni_session; # install by extension name, for the current active PG version
pig ext install omnigres; # install via package alias, for the active PG version
pig ext install omni_session -v 17;   # install for PG 17
pig ext install omni_session -v 16;   # install for PG 16
pig ext install omni_session -v 15;   # install for PG 15
pig ext install omni_session -v 14;   # install for PG 14
pig ext install omni_session -v 13;   # install for PG 13
```
```bash tab="yum" 
dnf install omnigres_17;
dnf install omnigres_16;
dnf install omnigres_15;
dnf install omnigres_14;
dnf install omnigres_13;
```
```bash tab="apt" 
apt install postgresql-17-omnigres;
apt install postgresql-16-omnigres;
apt install postgresql-15-omnigres;
apt install postgresql-14-omnigres;
apt install postgresql-13-omnigres;
```
```bash tab="ansible"
./pgsql.yml -t pg_ext -e '{"pg_extensions": ["omnigres"]}' # -l <cls>
```

[**Create**](/docs/pgsql/ext/create) this extension with:

```sql
CREATE EXTENSION omni_session CASCADE;
```
