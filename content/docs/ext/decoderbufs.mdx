## Download

To add the required PGDG / PIGSTY upstream repository, use:

```bash tab="pig"
pig repo add pgdg -u    # add PGDG repo and update cache (leave existing repos)
pig repo add pigsty -u  # add PIGSTY repo and update cache (leave existing repos)
pig repo add pgsql -u   # add PGDG + Pigsty repo and update cache (leave existing repos)
pig repo set all -u     # set repo to all = NODE + PGSQL + INFRA  (remove existing repos)
```
```bash tab="pigsty"
./node.yml -t node_repo -e node_repo_modules=node,pgsql # -l <cluster>
```

## Install

[**Install**](/docs/pgsql/ext/install) this extension with:

```bash tab="pig"
pig ext install decoderbufs; # install by extension name, for the current active PG version

pig ext install decoderbufs -v 17;   # install for PG 17
pig ext install decoderbufs -v 16;   # install for PG 16
pig ext install decoderbufs -v 15;   # install for PG 15
pig ext install decoderbufs -v 14;   # install for PG 14
pig ext install decoderbufs -v 13;   # install for PG 13
```
```bash tab="yum" 
dnf install postgres-decoderbufs_17*;
dnf install postgres-decoderbufs_16*;
dnf install postgres-decoderbufs_15*;
dnf install postgres-decoderbufs_14*;
dnf install postgres-decoderbufs_13*;
```
```bash tab="apt" 
apt install postgresql-17-decoderbufs;
apt install postgresql-16-decoderbufs;
apt install postgresql-15-decoderbufs;
apt install postgresql-14-decoderbufs;
apt install postgresql-13-decoderbufs;
```
```bash tab="ansible"
./pgsql.yml -t pg_ext -e '{"pg_extensions": ["decoderbufs"]}' # -l <cls>
```

Extension `decoderbufs` [**does not need**](/docs/pgsql/ext/create#list-of-extensions-without-ddl) `CREATE EXTENSION` command.
