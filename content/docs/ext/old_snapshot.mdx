## Download

This extension is built-in with PostgreSQL and does not need separate download.

------

## Install

Extension `old_snapshot` is PostgreSQL Built-in [**Contrib**](/docs/list/contrib) Extension which is installed along with the kernel/contrib.

[**Create**](/docs/pgsql/ext/create) this extension on PostgreSQL database with:

```sql
CREATE EXTENSION old_snapshot;
```
