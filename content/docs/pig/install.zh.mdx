## 脚本安装

安装 `pig` 最简单的方式是运行以下安装脚本：

```bash tab="默认"
curl -fsSL https://repo.pigsty.io/pig | bash
```
```bash tab="镜像"
curl -fsSL https://repo.pigsty.cc/pig | bash
```

该脚本会从 pigsty [软件仓库](/zh/repo/) 下载最新版 `pig` 的 RPM / DEB 包，并通过 `rpm` 或 `dpkg` 进行安装。

## 通过 APT / YUM 安装

`pig` 软件包同样可通过 [`pigsty-infra`](/zh/repo/infra) 仓库获取。
你可以将该仓库添加到系统后，使用操作系统的包管理器进行安装：

```bash tab="yum"
sudo tee /etc/yum.repos.d/pigsty-infra.repo > /dev/null <<-'EOF'
[pigsty-infra]
name=Pigsty Infra for $basearch
baseurl=https://repo.pigsty.io/yum/infra/$basearch
enabled = 1
gpgcheck = 0
module_hotfixes=1
EOF

sudo yum makecache;
sudo yum install -y pig
```

```bash tab="apt"
sudo tee /etc/apt/sources.list.d/pigsty-infra.list > /dev/null <<EOF
deb [trusted=yes] https://repo.pigsty.io/apt/infra generic main
EOF

sudo apt update;
sudo apt install -y pig
```

## 自我升级

安装完成后，你可以通过如下命令将 `pig` 升级到最新版：

```bash
pig update   # 将 pig 自身升级到最新版
```

## 源码构建

你也可以通过源码自行构建 `pig`。`pig` 是用 Go 语言开发的，源码托管在 [GitHub](https://github.com/pgsty/pig)

- https://github.com/pgsty/pig

```bash
git clone https://github.com/pgsty/pig.git
cd pig
go build
```
