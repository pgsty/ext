# DEB Extension Packages

There are **410** extensions available on Debian/Ubuntu compatible systems, **13** of them are Debian exclusive, and missing **13** EL exclusive extensions.

There are **71** built-in [**contrib**](contrib) extensions, in addition to **103** deb extensions provided by PGDG YUM repository, and **236** extensions provided by Pigsty.

There are **400** extensions available in the current major version PostgreSQL 16, and **397** ready for the latest PostgreSQL 17.
